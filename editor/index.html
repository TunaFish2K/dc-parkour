DC Parkour地图编辑器

编辑器使用说明

移动视角： 使用 I（上移），J（左移），K（下移），L（右移） 来调整视角。

添加边： 按下 Enter 键在画布中央添加一条新边。

选择边： 按 空格键/E 在已添加的边之间循环选择，当前选中边将以红色显示。

移动边： 使用 W（上移），A（左移），S（下移），D（右移） 来调整选中边的起始位置。

对齐边： 按下 Y 把当前边Y坐标对齐到300。

调整长度： 使用 下箭头（缩短）和 上箭头（加长）来调整边的长度。

调整面向： 使用 右箭头（顺时针旋转）和 左箭头（逆时针旋转）来调整边的面向角度。

预设方向切换： 按下 Tab 键切换至最近的预设方向（例如：0, 1/4π, 1/2π 等）。

复制边： 按下 V 键复制当前边。

生成反向边： 按下 F 键添加反向边。

虚化边： 按下 Q 键切换当前边的虚化状态。

删除平台： 按下 R 键删除当前选中的平台。

撤销操作： 按下 Ctrl + Z 组合键撤销上一步操作。

复制数据： 按下 C 键将将地图复制到剪贴板。

输出数据： 按下 P 下载地图。
